# Askademic AI

An adaptive mathematics learning platform designed to help students learn through direct answers, step-by-step explanations, and guided hints.

---

## Purpose

Askademic AI was created to support students who need additional assistance when learning mathematics.

Rather than simply providing answers, the platform aims to encourage understanding by offering multiple levels of support depending on the student's needs.

The project was independently developed by Vincent Lee and continues to evolve through teacher and student feedback.

---

## Features

### Direct Answer Mode
Provides concise solutions to mathematical problems.

### Step-by-Step Mode
Breaks solutions into logical stages to support understanding.

### Hint Mode
Guides students towards answers without immediately revealing the solution.

### Algebra Support
Supports:
- Solving equations
- Simplification
- Expansion
- Factorisation
- Algebraic manipulation

### Learning-Focused Design
Built to encourage independent problem solving rather than answer copying.

---

## Feedback and Testing

The platform has been reviewed by mathematics teachers and tested by students.

Feedback has been used to:
- Improve explanation quality
- Improve usability
- Improve mathematical accuracy
- Refine learning features

Planned improvements include:
